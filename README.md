# Scientific Android Calculator 📱📐

A beautiful, responsive, and lightweight Scientific Calculator application built for Android using Java.

---

## Features ✨

* **Basic Operations:** Addition, subtraction, multiplication, division, and percentage calculations.
* **Scientific Functions:** Trigonometric functions (`sin`, `cos`, `tan`), inverse trigonometric function (`arctan`), and factorials (`x!`).
* **Real-time Preview:** Displays the evaluated result dynamically as you type the expression.
* **Smart Input Validation:** Automatically handles operator replacements, decimal point checks, sign toggles (`±`), and balances brackets before calculation to prevent crashes.

---

## Technical Stack 🛠️

* **Language:** Java (JDK 17)
* **Architecture:** XML-based layout with `MainActivity` logic
* **UI styling:** Google Material Design Components
* **Build System:** Gradle (Kotlin DSL)

---

## How to Build & Run 🚀

### Prerequisites
* **Android Studio** (Koala or newer recommended)
* **Android SDK** (API 24+)
* **JDK 17** (or bundled with Android Studio)

### Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/stephengade/android-java-calculator.git
   cd android-java-calculator
   ```
2. **Open in Android Studio:**
   * Open Android Studio -> *File* -> *Open* -> Select the project folder.
   * Allow Gradle to sync.
3. **Run on Emulator or Device:**
   * Connect a physical device with USB debugging enabled, or start a virtual device (AVD) from the **Device Manager**.
   * Click the green **Run** (Play) button in Android Studio.
